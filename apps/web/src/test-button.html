Debugging Steps

Common Issues with React Native Web Buttons:

  • CSS pointer-events: none blocking clicks
  • TouchableOpacity event handling differences
  • Missing onPress handlers
  • Z-index stacking issues
  • Absolute positioned overlays blocking clicks

Console Debug Commands

Paste these in the browser console on your deployed site:

// Check if buttons exist
document.querySelectorAll('[role="button"]').forEach((btn, i) => {
    console.log(`Button ${i}:`, btn);
    console.log('Computed styles:', window.getComputedStyle(btn));
});

// Check pointer events
document.querySelectorAll('[role="button"]').forEach((btn, i) => {
    const styles = window.getComputedStyle(btn);
    console.log(`Button ${i} pointer-events:`, styles.pointerEvents);
    console.log(`Button ${i} cursor:`, styles.cursor);
});

// Test click programmatically
document.querySelectorAll('[role="button"]')[0]?.click();
